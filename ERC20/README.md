# ERC20

## Setup

```bash
$ npm install
```

## Compile

```bash
$ npx hardhat compile
```

## Test

```bash
$ npx hardhat test
```

## Deploy(Testnet)

```bash
$ npm run deploy:goerli
```

## Deploy(Notice Slack)

```bash
$ npm run deploy:slack
```
